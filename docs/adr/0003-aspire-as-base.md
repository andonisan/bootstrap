# 3. Aspire como base tecnológica

Date: 2025-05-25

## Status

Accepted

## Context

Necesitamos simplificar el desarrollo local, la observabilidad y la telemetría sin añadir complejidad innecesaria.

## Decision

Usaremos .NET Aspire como base tecnológica para nuevos proyectos. Aspire ofrece dashboard integrado, telemetría out-of-the-box y simplifica la observabilidad local.

## Consequences

✅ Menor esfuerzo inicial en setup y observabilidad  
✅ Dashboard integrado  
❌ Dependencia tecnológica adicional (Aspire)
